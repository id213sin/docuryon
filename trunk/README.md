# Documents

Welcome to Docuryon! This is your document storage folder.

## Getting Started

Simply add your documents to this `trunk/` folder. The explorer will automatically display them.

### Supported File Types

- **Documents**: Markdown (.md), Text (.txt), HTML (.html)
- **Images**: PNG, JPG, JPEG, GIF, SVG, WebP
- **PDFs**: PDF documents with preview support
- **Code**: JavaScript, TypeScript, Python, JSON, CSS, and more

### Folder Organization

You can create unlimited nested subfolders to organize your documents:

```
trunk/
├── projects/
│   ├── project-a/
│   │   └── docs/
│   └── project-b/
├── images/
├── reports/
└── notes/
```

## Features

- **Grid & List Views**: Toggle between views using the toolbar
- **Keyboard Navigation**: Use arrow keys to navigate
- **File Preview**: Click on a file to see its preview
- **Search**: Use the search bar to filter files
- **Theme Support**: Switch between light, dark, and system themes
